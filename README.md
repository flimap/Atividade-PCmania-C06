# PCMania - Sistema de Compra de Computadores

Projeto desenvolvido em Java com o objetivo de simular um sistema de compra de computadores, seguindo os princípios de Programação Orientada a Objetos (POO) e as especificações propostas em atividade acadêmica.

# Objetivo do Projeto
Desenvolver um sistema que permita ao usuário selecionar computadores a partir de promoções pré-definidas, adicionar múltiplos itens ao carrinho, visualizar as configurações dos produtos adquiridos, calcular o valor total da compra e simular o envio do pedido.

# Estrutura do Projeto
O sistema foi estruturado utilizando as seguintes classes:
Cliente, Computador, HardwareBasico, SistemaOperacional, MemoriaUSB, ProcessarPedido e Main.

# Desenvolvimento e Construção do Código
A implementação do projeto foi realizada de forma incremental, com a estrutura geral das classes sendo construída inicialmente de forma independente, seguindo o diagrama UML proposto.

Durante o desenvolvimento, a lógica principal e a modelagem das classes foram elaboradas de forma própria, incluindo:
- Definição de atributos e construtores
- Implementação de arrays para armazenamento de objetos
- Criação de métodos principais como adição de computadores e cálculo de total
- Estruturação do menu interativo com entrada de dados via Scanner

# Apoio e Ajustes Realizados
Durante o desenvolvimento do projeto, foi utilizada a ferramenta ChatGPT, baseada no modelo GPT-5.3, com o objetivo de auxiliar na validação, correção e refinamento de trechos específicos do código.

Os prompts utilizados tiveram como foco principal:
- Verificação da estrutura de classes e métodos
- Correção de erros lógicos em loops e manipulação de arrays
- Ajustes na implementação de métodos (como cálculo de total e exibição de dados)

Exemplos de prompts utilizados:
- "Está correta a estrutura da classe Cliente?"
- "Como posso calcular o total da compra percorrendo um array?"
- "Essa lógica com do-while funciona para múltiplas compras?"
- "Como evitar erro ao acessar posições nulas no array?"

Os resultados obtidos foram satisfatórios, contribuindo principalmente para a identificação de erros pontuais e melhoria da organização do código. A lógica principal, modelagem das classes e implementação geral foram desenvolvidas de forma autoral, sendo a IA utilizada como ferramenta de apoio para revisão e ajustes.

# Funcionalidades Implementadas
- Interface simples via terminal para seleção de produtos
- Menu interativo com repetição até finalização da compra
- Armazenamento de múltiplos computadores em array
- Exibição das configurações dos computadores adquiridos
- Cálculo do valor total da compra
- Exibição dos dados do cliente ao final
- Simulação de envio do pedido

# Considerações Finais
O projeto demonstra a aplicação prática de conceitos fundamentais de programação orientada a objetos, incluindo encapsulamento, associação entre classes e separação de responsabilidades.

O processo de desenvolvimento envolveu construção ativa do código, identificação de erros, testes e ajustes, resultando em um sistema funcional e aderente às especificações propostas.
